# 第一卷《周秦卷》Agent 入口

先读 `../../SERIES_SPEC.md` 和 `../../AGENTS.md`；根规范是本卷的生产、
角色、审阅、构建与发布标准。本文件只定义周秦卷的例外。

## 范围与结构

- 范围为西周至秦亡；秦末起事仅作为秦帝国终局和汉卷开端的衔接。
- 账本位于 `data/event-ledger/western-zhou.csv`、`spring-autumn.csv` 与
  `warring-qin.csv`；总目标为 1,300--1,900 条，不能以正文篇幅替代覆盖率。
- 正文依 `western-zhou/`、`spring-autumn/`、`warring-qin/` 分期；共享年序
  位于 `common/`，国家 dossier 位于 `states/`，跨国问题位于 `synthesis/`。
- 春秋须分别覆盖周王室、齐、晋、楚、秦、鲁、郑宋、吴越；战国须分别覆盖
  秦、楚、齐、燕、韩、赵、魏与残余周室。跨国人物、变法、战争与外交必须
  另写合卷并回链事件。

## 材料与争议

- 基础材料为《尚书》《春秋》《左传》《国语》《战国策》《史记》、竹书
  纪年、金文、简牍和考古；以材料性质而非固定等级交叉。
- 特别核验牧野与共和年代、周公摄政、镐京陷落、春秋战国分期、商鞅、长平、
  六国统一和秦亡；不将后出叙事、兵数或人物说辞写成无争议事实。
- 《过秦论》等汉人回望材料可用于古文窗口，但不作秦朝现场记录。
