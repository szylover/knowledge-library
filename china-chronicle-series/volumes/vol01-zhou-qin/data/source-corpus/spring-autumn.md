# 春秋史料研究矩阵

## 使用范围与证据标记

本矩阵服务于前771—前473年的春秋材料池，并须同 `data/event-ledger/spring-autumn.csv` 的事件 ID 一起使用。现账本有 **75** 条；它是当前正文的选题清单，不是时代事件总数，也不是材料可信度的替代物。

写作及账本注记应先标明材料层次，再说明可作何种判断：

| 标记 | 可作为的证据 | 不能据此推出的内容 |
|---|---|---|
| **经年** | 传世《春秋》所保存的鲁国年次、行动者、战争/会盟/弑君等极简事实 | 鲁国外的完整局势、动机、战术、谈判原话 |
| **叙事** | 《左传》《国语》保存的事件顺序、政治记忆、礼仪和论辩框架 | 对白、预言、心理和因果链的逐字实录 |
| **后设** | 《史记》、经传解释及后出的综合史书怎样整理早期传统 | 春秋现场的单一视角，或无争议的精确日期 |
| **物证** | 器物铭文、盟书、城址、墓葬、兵器和区域调查可确认的称谓、技术、空间或精英网络 | 传世故事中的谋议、人物品格、某次战役每一步 |
| **解释** | 现代历史地理、考古、制度和环境研究提出的模型 | 直接等同于古人动机，或填补失传的逐年档案 |

“经年”不等于保存了春秋时的原始档案；它是传世文本所保留的鲁国编年传统。若同一事件仅有后出叙事，正文须写明“传世叙事称”或“后来的《史记》将其组织为”，不可写成已由同时代材料独立证实。

## 传世文本：层次、覆盖与交叉方法

| 材料 | 年代/问题覆盖 | 最适合确认或追问 | 文本层与使用边界 |
|---|---|---|---|
| 《春秋》 | 前722—前481，以鲁君年次排列 | 会盟、伐战、来朝、继承、灾异及部分结果的编年骨架 | 鲁国中心且极简；年次、主体和用词须先核，不能由沉默推出未发生 |
| 《左传》 | 覆盖《春秋》年代，尤长于外交、战争、卿族与礼仪 | 事件关联、当时人可使用的政治语言、叙事版本的比较 | 成书、编纂层次与材料来源均有争议；是后出叙事，不是会议录或战地记录 |
| 《公羊传》《谷梁传》 | 解释《春秋》的义例与政治意义 | 经学接受史、经文何以被读成褒贬 | 不可用其义例代替事件本身的独立证据 |
| 《国语》 | 周、鲁、齐、晋、楚、吴、越等国的言说传统 | 国别政治记忆、说辞如何塑造合法性 | 国别说辞体，年代和场景常须与《春秋》《左传》交叉；不可按逐字发言引述 |
| 《史记》本纪、世家、列传 | 贯通周、列国与人物长线 | 国别序列、不同传统的汉代汇编、与经传异同 | 汉代回望，常将复杂过程写成兴亡和人物线；明示为后设叙事，不倒灌为现场事实 |
| 《竹书纪年》 | 西周末、东周及战国的异年表线索 | 与传世年表的异同、经传不见的线索 | 今本、古本及重建问题突出；须注明版本和异年，不能单独裁定年代 |
| 《管子》《论语》《孙子兵法》及相关传说 | 管仲传统、晚春秋思想、兵学接受 | 后世如何理解政治经济、礼与兵；可与材料史相互提问 | 均有复杂成书/传本层次；不把《管子》制度清单、孔子后世形象或孙武传记直接安放进某一年政策或战役 |

### 推荐核对顺序

1. 先以《春秋》定鲁历年、事件名称及最小可证事实；前771—前723和鲁国经文不及处，明记缺口，而非假装有同等编年。
2. 用《左传》与《国语》比较事件顺序、参与者和说辞；两者相合仍只是传世叙事相合，不自动升级为同时代记录。
3. 用《史记》世家、本纪、列传检查国别长线和异文；遇到更完整的人物故事，标为汉代整合。
4. 年代、地望、族群或制度有争议时，列出《竹书纪年》、金文、考古与现代考订的分歧，不以一个整齐故事消除差异。

## 出土与物质材料矩阵

| 材料群 | 可贡献的事实类型 | 优先对应的问题 | 关键限制 |
|---|---|---|---|
| 青铜器铭文（含王命、赏赐、盟誓与族氏材料） | 称号、受命、赏赐、宗族和精英交往的物质痕迹 | 周王室名分、诸侯/卿族网络、礼器与政治资源 | 年代、出土地和传世真伪须逐器核查；器铭通常不能复原一次会盟或战役全程 |
| 侯马盟书及同类盟誓材料 | 春秋末晋系精英间结盟、排斥与誓约的书写实践 | 晋卿族、家臣和盟约的实际运作 | 属特定地点与关系网络，不能泛化为所有国家或所有会盟的格式 |
| 城址、聚落、道路与区域调查（洛阳王城、临淄、曲阜、晋都、楚都及江淮区域） | 城邑尺度、空间层级、交通和环境约束 | 王畿资源、齐晋楚腹地、吴越水网、战场地望 | 遗址年代常为较宽区间；城市存在不证明某年由谁控制，也不证明传世战争路线 |
| 墓葬、车马坑、兵器与铸造遗存 | 精英等级、兵器技术、车战/水陆动员的物质条件 | 军事供给、礼制、楚与吴越物质文化 | 随葬品不等于全社会消费；技术能力不自动说明战场部署或国家政策 |
| 带铭越王兵器及吴越金属器 | 君主称号、下游精英工艺与政治象征 | 吴越国家形成、兵器与礼仪物质性 | 只能旁证政治文化与铸造能力，不能证明《国语》《史记》中的谋臣计策、苦行或战场对白 |
| 简帛与传世文本的出土校勘材料 | 战国—汉代以前文本流传、篇章异文及概念史 | 《孙子》文本史、经传/思想材料的层次 | 多数并非春秋当年行政档案；早期文本存在不等于传说人物必参与某次事件 |

考古材料进入正文或账本时，应写明器物/遗址名称、出土与年代依据、能回答的问题和不能回答的问题。没有可靠发掘背景的市场器物、未核实铭文释读及“以器证人品”的说法不得作为核心证据。

## 按议题建账与写作指引

### 国家、继承与卿族

- **优先链**：本国《春秋》可见年次 → 《左传》相关年 → 《国语》国别说辞 → 《史记》世家 → 金文、盟书、城址与现代制度研究。
- **账本应拆开**：君主死亡/即位、出奔、弑君、卿族任军、封邑或家臣冲突、周室册命，避免以“某氏崛起”吞掉几十年过程。
- **写作警报**：田氏、三桓、晋卿、楚令尹和郑执政并非现代官僚党派；封邑、宗族、军权与职位的实际组合多不能逐年复原。把“资源可继承”作为解释，须同具体事件分开标为解释。

### 外交、会盟与战争

- **优先链**：经文年次和参战者 → 《左传》叙事 → 《国语》说辞 → 《史记》整合 → 历史地理、城址/道路与兵器材料。
- **账本应拆开**：会盟的召集、参会/不参会、援军、围城、和解、背盟和后续从属；会盟不可只作为“霸权建立”的装饰性节点。
- **写作警报**：盟辞、使者机锋、退三舍、烛之武、问鼎、泓水与黄池争先，首先是叙事材料；地形能解释可行性，不能单独证明伏击、行军路线或唯一战略意图。

### 经济、劳役与社会

- **优先链**：战争、迁徙、城筑、救荒的经传记载 → 金文、城址、墓葬、生产/交通考古 → 区域环境与经济史研究。
- **可稳妥提出**：长期动员需要粮秣、道路、城邑和劳役；封邑及家臣网络与资源归属有关；秦输粟、迁邢、围郑、远征殽等事件可作为可见切口。
- **不得越界**：传世材料没有完整赋税簿、户籍、粮仓清册或人口统计。不能把战国或汉代的盐铁、货币、县制、亩制和统一税制倒灌到齐桓、子产或春秋任何一国；《管子》的精密经济方案尤其只能作为后起文本传统。

### 礼制、文化与思想

- **优先链**：《春秋》文本与三传的不同读法 → 《左传》《国语》的礼仪叙事 → 《论语》及相关早期文本的传本层次 → 铭文、墓葬与现代思想/礼制研究。
- **可稳妥提出**：周王室名分仍是可被争用的政治语言；《春秋》在后世经学中获得的意义不同于其最小编年信息；孔子及儒家传统应置于晚春秋至战国的文本形成与接受史中。
- **不得越界**：不把“孔子删《春秋》”、一则礼仪对白、后世《公羊》义例或墓葬等级当作已被独立确证的春秋现场思想。文化变化通常缺逐年证据，宜用较宽时段和“材料显示/后世解释”的措辞。

## 现阶段关键缺口与提交检查

1. **前750—前726空带**是最明显的编年断裂；东迁后几十年的周郑、鲁齐、晋、秦、楚事件须先建立低置信度但可追溯的候选清单。
2. **国别不平衡**：齐、晋、秦、吴越已有较多名场面，宋、卫、陈、蔡、曹、许及戎狄等群体仍多作为大国叙事的背景；补入其迁徙、朝聘、从属转换和受战影响的事件。
3. **中段连续性**：前625—前576的楚郑、前575—前501的齐/晋卿族与周室内战，仍需把战争之间的继承、会盟、城邑和资源节点补齐。
4. **末段衔接**：前475—前473只有越灭吴；须说明吴亡年次的版本差异，并以事件链而非“春秋在前481年突然结束”连接到战国账本。
5. **慢变量不足**：经济社会与文化思想正文已有解释性框架，却缺可入账的物质材料和区域研究索引；在无新材料前，不应伪造精确年度事件。

每新增或扩写一个春秋事件，至少检查：①最小可证的日期/顺序和主体；②一条基础文本或物证；③该材料属于经年、叙事、后设、物证还是解释；④一项可核的现代研究方向（历史地理、铭文、考古、制度史或文本史）；⑤是否把后出故事、解释或考古旁证误写成直接证据。
