# 史料池与验收矩阵

本目录把第一卷的材料性质、可核查范围和账本覆盖分开记录。史料池不是可直接复制到正文的事实库：每一条账本记录都必须同时保留最小日期或相对次序、行动者、核心来源、`source_caveat` 与研究方向。

## 账本验收字段

三个 CSV 保留原有十个字段和全部行、ID、顺序，并追加两个非空字段：

| 字段 | 用途 |
|---|---|
| `source_caveat` | 说明核心材料的文体、保存距离或物质证据的外推限制；不以材料并列冒充独立确证。 |
| `tier` | `A` 为改变政权、制度、战争或帝国结构的指定转折；`B` 为国家线不可缺的政治、军事、继承或资源节点；`C` 为维持年序和因果连续的桥接节点。分级不改变事件可信度，也不替代正文深描时的复核。 |

`source_caveat` 依材料组合归入《春秋》经年、传世叙事、后出史学、传世文献、出土/物证交叉或一般最小证据六类；所有行均有值。`tier` 是本次账本验收的编辑优先级，不将后出叙事或不确定年代升级为事实。

## 矩阵导航

- `western-zhou.md`：西周材料、王年与金文问题。
- `spring-autumn.md`：春秋经年、传世叙事与物证矩阵。
- `warring-qin.md`：战国、秦帝国和秦亡的材料矩阵。
- `coverage-actioner-gap-matrix.md`：按时间带、行动者与材料限制列出的覆盖计数和未完成项。
