# 周秦争议账本

**用途。** 本账本只登记会改变周秦叙事读法的争议，不为任一解释裁决胜负。它把现有事件账本、已写正文和材料类型并置，供后续写作保留可说范围与证据缺口。

**定位规则。** 每项的“账本记录”均采用 `data/event-ledger/<file>.csv#<id>`；“已写路径”是现有正文的相对路径，不意味着该正文已经解决争议。材料层以来源矩阵的“经年／叙事／后设／物证／解释”区分，并对应 evidence-passport schema 的 `annalistic_record`、`narrative_text`、`retrospective_history`、`excavated_text_or_inscription`、`archaeological_context`、`modern_research` 等类型。

**证据护照状态。** 已核对 `shared/evidence/evidence-passport.schema.json` 及示例；当前卷没有可与下列事件 ID 对应的单项 evidence passport。因此本文件不把“材料层”伪装成护照，也不新增或改写事件账本。

## CL-01 牧野的绝对年代与战场叙事

- **账本记录：** `data/event-ledger/western-zhou.csv#WZ-1046-MUYE`
- **已写路径：** `chapters/western-zhou/states/royal-court/foundation-and-regency.tex`
- **问题：** “约前1046年”如何成立？利簋的甲子日、传世的牧野叙事和灭商的具体战场过程能否合成一份精确战报？
- **竞争材料／层次：** 利簋是近时金文的物证；《尚书·牧誓》是传世政治文本；《史记·周本纪》是汉代回望；王年和日食等综合推算属于现代解释。它们回答的不是同一个问题。
- **可说：** 周取代商是有金文与传世传统共同支撑的重大断裂；前1046年是现行的综合估算，利簋使“甲子朝、征商”成为不可忽略的时间锚点。
- **不可说：** 不把该年写成没有争议的精确年代，不由甲子日复原交战地点、兵数、阵势或所有人物言辞，也不把后出的正当化叙事当作同时代战报。
- **可收窄的证据：** 出土背景明确且可互校的早周铭文、可重复的年代学模型，以及与考古地层相连的商周转折材料可检验绝对年代；它们仍未必能补足战场细节。

- **竞争材料／层次：** 《尚书》的《金縢》《大诰》《康诰》等篇章有复杂成文、传本与政治语境；《史记》加以王朝化整合；大盂鼎等金文能旁证处分、册命和政治网络，却不提供完整年表。三类材料互有交集而不能互补成会议记录。

## CL-03 共和：连续纪年的锚点，还是可定名的摄政体制？

- **竞争材料／层次：** 《国语·周语上》提供危机与周召共和叙事；《史记·周本纪》延续这一组织；《竹书纪年》保存共伯和“干王位”的异说；西周晚期金文和现代年代研究只能从旁检验制度与年代，不能直接命名执政者。
