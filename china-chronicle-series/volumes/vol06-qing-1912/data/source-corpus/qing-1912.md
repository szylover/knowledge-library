# 第六卷来源矩阵：清（1644—1912）

## 使用范围、取证与账本规则

| 项目 | 本卷规定 |
|---|---|
| 覆盖 | 自清军入关及北京政权建立（1644）至宣统帝退位（1912）。明清鼎革、南明、三藩、郑氏台湾与准噶尔史事按与清朝形成相连的时段书写；革命、地方独立与民国临时政府只覆盖其导致帝国终结的部分。清朝不是单一“中原王朝”的同义词：满洲、蒙古、新疆、青藏高原、东北、西南、台湾、海疆和海外交往均有自己的材料节奏。 |
| 账本用法 | 每一事件先以可回查的近时编年、档案或相互独立的同时代材料确立日期/相对序列；再按问题补入制度、财政、地方、外文、物质或社会材料。`core_sources` 须写书名与卷/年/月日、档案全宗—件号、报刊日期版面、日记日期、文书编号或发掘报告/图版；`source_caveat` 写明成书距离、文体、语言、产生机构、利益位置和主要异说。 |
| 纪年 | 保留清帝年号、干支、月日和换算依据。涉及南明、郑氏、朝鲜、日本、俄历、格里历、伊斯兰历、藏历或地方纪年时，原始纪年与换算日期并列。改元、闰月、时区、报刊刊期或电报发出/到达日期有冲突时用“约”或分别列项，不能制造精确日序。 |
| 版本与检索 | 传世史书优先使用有校勘记的可靠整理本；档案引用影印件/原件与档号；报刊标明报名、日期、版面；条约同时核对签署文本、批准/换约文本及至少一方外交档案。数据库、全文库和影像平台仅为检索入口，不替代可核版本、档号或正式发掘报告。译文应说明译者、底本与原文语言。 |
| 不能直接推出 | 奏折、上谕、章程、律例、条约和改革方案能证明呈报、命令或规范，不自动证明地方执行；户口、田赋、海关、军额、战果、灾荒和贸易数字必须附统计机关、地域、单位、口径和政治场景；一地档案、一家日记、一份传教报告或一处遗址不能代表全帝国。官修的“逆”“匪”“蛮”“回”“番”等分类和外文的文明/帝国话语均不得直接变成群体本质或动机。 |

## 九、进入事件账本前必须处理的争议与空白

| 主题 | 不可预设的结论 | 最低处理标准 |
|---|---|---|
| 清初征服与鼎革 | 1644 即代表全国统一，或“入关/亡明”可解释所有地区的政治选择 | 分列北京政权、南明、郑氏、三藩、地方抵抗与各区域纳入的时间；将清廷、地方和对手叙述并置，战争伤亡仅报告可追溯口径 |
| 皇权、奏折与中央控制 | 朱批或上谕即说明君主知道全情、命令已执行或地方完全服从 | 标明信息来源、批示、转发、部议、地方执行和结果；利用地方档、账册或对方记录检验文书链的断点 |
| 人口、田赋、财政与“盛世” | 户口、垦田、库银和粮价数字可以直接比较为帝国真实总量 | 每项随附年份、地域、统计单位、单位换算与政治背景；区分登记人口、赋役单位和实际人口，拒绝无口径的跨朝排名 |
| 战争、叛乱与族群暴力 | 方略、战报或外国军报中的兵数、歼获、伤亡及族群标签即为客观事实 | 将数字标为报告者的声明，寻找补给、墓地、地方/对方记录等约束；使用当时原词并区别政治归属、语言、宗教、法律身份与自称 |
| 条约、主权与边界 | 条约单一译本、朝贡称谓或一张后制地图足以确定主权、边界和实际治理 | 核对多语文本、批准状态、签署与执行过程，并加入地方、邻国和地图制作语境；图上界线仅作约略解释，不充当主证 |
| 洋务、改革与现代化 | 设立机构、购置机器、颁布学制即可证明全国现代化或失败 | 区分方案、资金、人员、产出、地方采用与持续时间；海关/企业/地方资料只能说明其覆盖范围内的变化 |
| 宗教、教案与秘密结社 | 官府的“邪教/匪”或传教士的“迫害”标签足以说明群体性质和冲突原因 | 将教义、组织、地方纠纷、土地/税役、外交保护和暴力结果分开；至少并读官方、教会或地方社会材料 |
| 1911—1912 与帝国终结 | 单一革命中心或退位诏书即可解释全国政治转换 | 将各省独立、军政控制、财政交通、谈判和退位分列；报刊/电报时间与地方落实须分开，保留未能确定的地方时序 |

## 十、交接给账本作者的最小字段清单

每一候选事件至少提供：

1. 原始日期（年号/干支/月日或外文历法）、换算日期及换算依据；
2. 行动者的当时称谓、政权/机构、地点及地名规范化说明；
3. 一项主证（精确卷页、档号、报刊版面或文书/报告编号）和一项适合该问题的交叉材料；
4. 事实层级：命令、呈报、谈判文本、观察、实施、结果或后出记忆；
5. 语言、作者/机构、成文时间与观察位置；
6. 对数字、疆界、战果、族群分类、动机或日期的明确保留；
7. `source_caveat` 中一句可执行的限制说明，而非笼统写“史料有偏见”。

本矩阵是账本前的取证路线，不替代逐条核读原文、档案或正式考古报告。材料密度最高的北京、江南和条约口岸不应决定全卷叙事比例；缺少本地语言或基层材料的区域，应将沉默写入证据判断，而不是由中央叙事补足。
