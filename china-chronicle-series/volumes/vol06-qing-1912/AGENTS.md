# 第八卷《清至1912》Agent 入口

先读 `../../SERIES_SPEC.md`、`../../AGENTS.md` 与
`data/source-corpus/qing-1912.md`。根规范是本卷的生产、角色、审阅、构建与
发布标准；本文件只定义本卷例外。

## 范围与结构

- 范围为1644--1912年，从清军入关和北京政权建立至宣统帝退位。南明、三藩、
  郑氏台湾、准噶尔、革命和地方独立只写入与清帝国形成或终结直接相连的部分。
- 事件先入 `data/event-ledger/qing-1912.csv`。保留清帝年号、干支、月日及
  换算；南明、郑氏、朝鲜、日本、俄历、格里历、伊斯兰历、藏历或地方纪年
  与换算日期并列。
- 满洲、蒙古、新疆、青藏高原、东北、西南、台湾、海疆和海外交往都是独立
  的行动空间；不得压缩为单一“中原王朝”叙事。

## 材料与争议

- 以《清实录》《清史稿》、奏折、军机处档、内阁大库、地方档案、报刊、
  条约、外文档案、满蒙藏维文材料与考古交叉。
- 奏折、上谕、章程、律例和条约只能证明文书或规范；户口、田赋、海关、
  军额、战果、灾荒与贸易数字必须注明机构、地域、单位、口径和政治场景。
- 禁止把“逆”“匪”“蛮”“回”“番”或外文帝国话语变为群体本质；边疆、
  宗教、条约、战争和辛亥革命须并列相关行动者与材料语言。

## 卷内交付

图件与专题要显示帝国的多区域治理、交通、财政、战事和信息网络；发布仍只
由 Integrator 依根规范同步至 Azure Blob。
