# 第四卷《两晋与南北朝》Agent 入口

先读 `../../SERIES_SPEC.md`、`../../AGENTS.md` 与
`data/source-corpus/jin-northern-southern.md`。根规范是本卷的生产、角色、
审阅、构建与发布标准；本文件只定义本卷例外。

## 范围与结构

- 范围为280--589年，从孙吴降服至隋军入建康、陈亡；280年前只作必要背景。
- 事件先入 `data/event-ledger/jin-northern-southern.csv`。该账本从母卷筛出
  876条记录；不得以批量 `event-anchors` 或 `remaining-ledger` 文件替代
  连续正文。
- 正文覆盖西晋、东晋与十六国、南北朝、东西魏至隋，以及社会、宗教、边疆
  与行政综合；不得修改母卷以绕开本卷结构。

## 深度与读者验收闸门

- 本卷二八〇至五八九年的多政权范围，成书时至少应有 **220 个实际物理页**；
  其中至少 **190 页** 必须是合格的连续读者正文页。计数须以最终 PDF 的物理页
  为准，不以字数、锚点、账本条目或 TeX 源文件行数替代。
- 封面、扉页、前言、读法说明、目录、纯图页、图件索引、事件／政权／主题／
  来源索引、附录与书系承接说明，均不得计入这 190 页。含图的正文页只有在
  图前后仍有足量、连续且不可由图件或导航替代的叙事时，才可计为读者正文页。
- 不得用空白、强制分页、重复段落、批量锚点、账本转录、索引、资料框或机械
  改写的桥接句填充页数。每一计入页数的段落须推进特定事件簇的时间、空间、
  行动者、资源选择与后续压力，并在必要处保留材料能证明与不能证明的界线。
- 验收前须分别报告：最终 PDF 的物理页数、可计入的连续读者正文页数、排除页
  的页码范围与理由；同时抽读西晋、东晋与十六国、南北朝、东西魏至隋和综合
  各至少一段开头、转折与结尾，确认多政权、迁徙、财政、宗教、边疆和地方生活
  均以非模板叙事接入年序。

## 材料与争议

- 以《晋书》、南北朝正史与《资治通鉴》维持年序，结合墓志、造像记、佛道
  文献、河西/吐鲁番文书、城址和现代研究。
- 特别说明十六国年号/疆域、南渡规模、北魏制度的地区执行、族群称谓、
  宗教政策、战争数字及589年后行政接收的证据边界。
