# 第二卷来源矩阵：汉至南北朝（前 206—589）

## 使用边界

| 项目 | 规定 |
|---|---|
| 覆盖 | 西汉建立至陈亡于隋；为解释断裂，可上溯秦末，延及隋的统一处置。三国以东汉末（约 184）至 280 为连续问题域；十六国与东晋、北魏并列，不作为“北朝的附录”。 |
| 用法 | 下列材料用于建立和逐条核验 `data/event-ledger/`，并为后续制度、社会、宗教及地方专题提供可追溯的出处。一个事件至少标明主证和交叉材料；材料相冲突时保留分歧。 |
| 禁止推论 | 后出编纂史书、墓志的追赠官衔、宗教传记的神异叙述、以及考古分期，均不可单独证明具体事件、人口总量或行动者动机。 |
| 版本与引注 | 正史优先使用中华书局点校本并核对校勘记；简牍、墓志、造像记和发掘材料引报告/图版/简号（或志号）及释文版本；电子数据库只能作检索入口，不替代可核查版本。 |

## 六、待在事件账本中显式标注的争议与空白

| 主题 | 不能预设的结论 | 最低处理标准 |
|---|---|---|
| 年代、年号与并立政权 | 十六国、东晋、北魏及南朝的纪年可以无缝互换；后出年表必然正确 | 同时保留原始纪年、换算日期、换算依据和冲突来源；无法裁定时写“约”并说明原因 |
| 户口、土地和财政数字 | 正史户数、口数、垦田、军粮即真实全国总量 | 记录原文口径、统计机构、地理范围和政治场景；只报告可比范围，拒绝跨朝直接排名 |
| 军事规模和战果 | 兵额、斩获、俘虏、死伤和疆域图可按史书数字照录 | 将数字视作声明，寻找补给、城址、文书或敌方记录的约束；无交叉证据则标“未核实” |
| 族群与“汉化” | 史书族名对应稳定、单一的现代民族；改姓/服饰/官制必然表示同化 | 使用当时原词，区分政治归属、语言、婚姻、法律身份和自我表述；不以单一材料定义群体 |
| 门阀、寒门与社会结构 | 士族传记即可代表社会，或一个墓志可证明整个阶层 | 把精英文本、墓志网络、基层文书和聚落材料分层；明确妇女、奴婢、军户、流民和边民的材料缺口 |
| 宗教政策与信仰 | 皇帝诏令、灭佛记载或高僧神异能直接说明社会整体信仰 | 分开国家法令、寺院财产、僧尼网络、造像/墓葬和文本修辞；标明文本成书与事件距离 |
| 制度起源与落实 | “均田”“三长”“九品”“占田课田”“府兵”等名称出现即全国同步施行 | 分别记载制度文本、首次可见实践、地区、对象与后世追述；制度专题不得替代逐年事件证据 |
| 考古代表性 | 一座大墓、一个都城发掘区或一批钱币足以说明全国经济与文化 | 引正式报告、明确样本边界和断代；把“发现”与“全域推断”分写 |
| 空间与边界 | 现代地图可精确绘出每年国界 | 只绘/记可证实的城、道、河、军镇与影响区；边界表述为约略并附时点和证据类型 |

## 七、账本接入规则

| 事件类型 | 至少应并列的材料 | 来源注记必须写明 |
|---|---|---|
| 改元、即位、废立、政变、战役、盟约 | 一种编年/本纪材料 + 同期或异政权材料；必要时加年号/天象校验 | 正统纪年采用何者；异说是否来自同一史料链；战果与兵数是否只是单方声明 |
| 政令、制度与财政措施 | 诏令/志书或传记 + 能显示执行、地域或受众的文书、碑刻、考古或现代专题研究 | 文本日期、适用对象与地域；“颁布”还是“可见执行”；是否为后出制度志追述 |
| 人口、赋役、土地、物价与生产 | 正史统计/志书 + 出土簿籍、钱币/仓储/生产遗存或方法明确的研究 | 数字口径、区域、年度、样本和不可比项；不得省略估计性质 |
| 宗教、思想、教育与文本 | 官修史、目录或诏令 + 宗教文本/碑刻/造像/寺址之一 | 作者、文本成书或抄写日期、事件日期及文体目的；神异或护教叙述是否仅作观念史证据 |
| 地方、族群、迁徙与边疆 | 中央史书 + 地方文书、墓志、区域史、邻国/多语材料或考古之一 | 地名定位、群体称谓的原文、材料语言和观察立场；不可由中央分类直接替代地方自称 |

> 本矩阵是来源选择和证据限度表，不是事件清单、年表或章节正文。后续账本条目应链接到具体卷、篇、简号、志号或发掘报告，不能只笼统标注“正史”或“考古”。
