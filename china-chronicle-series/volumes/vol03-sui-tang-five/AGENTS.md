# 第五卷《隋唐五代》Agent 入口

先读 `../../SERIES_SPEC.md`、`../../AGENTS.md` 与
`data/source-corpus/sui-tang-five.md`。根规范是本卷的生产、角色、审阅、
构建与发布标准；本文件只定义本卷例外。

## 范围与结构

- 范围为581--960年，从隋受禅至宋代取代后周；五代与十国并列叙述，不能以
  中原五朝替代吴、南唐、吴越、闽、南汉、楚、荆南、前后蜀、北汉等政权。
- 事件先入 `data/event-ledger/sui-tang-five.csv`，正文只能扩写已登记事件。
- 隋末群雄、吐蕃、回鹘、渤海、新罗、日本及西域政权按事件关联进入
  dossier 或跨政权专题，而不作唐朝的附属背景。

## 材料与争议

- 以《隋书》、两《唐书》、两《五代史》《资治通鉴》及《考异》为编年
  主干；制度结合《通典》《唐会要》与诏令，地方和边疆结合敦煌吐鲁番文书、
  碑志、城址和外部材料。
- 保留隋唐改元、闰月、藩镇/十国并行年号与外部纪年的换算依据；不得将
  制度条文、藩镇军数、族名、朝贡语言或考古个案外推为全境事实。

## 卷内交付

图件放入相应 `figures/` 子目录，显示运河、道路、边疆与多政权空间的材料
限度。完成任务时更新本卷进度与队列。

## 读者正文深度闸门

581--960年的跨度不得以少数名场面、图件或索引充数。声称本卷通过读者优先
验收前，Integrator 必须从构建日志和 PDF 逐页复核：全书至少 **220 个物理
页面**，其中至少 **190 个合格的连续读者正文页面**。后一个数字排除卷面、
“怎样读”、目录、扉页、纯图页、图表说明、索引、来源附录、书系承接、空白页
与为凑页而设的留白；章节开头、连续叙事中的必要信息框和紧邻正文且由正文解释
的图注可计入，但不能以信息框、锚点清单或重复段落替代 prose。

这 190 页须分布于隋、初唐、武周—开元、安史—晚唐、五代十国五个连续时段；
每段都要在具体事件场景中交织多政权、地方与财政、道路/边疆、宗教或社会生活，
并以本卷来源矩阵规定的正史、编年、文书、碑志、考古或外部材料说明可说范围。
达到数字仍须抽读反模板、事件锚点、读者路径和跨索引路线；任一项不通过，不得
发布 PDF 或同步 Blob。
