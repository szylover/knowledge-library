# 第四卷来源矩阵：宋、辽、西夏、金、元（960—1368）

## 使用范围与证据规则

| 项目 | 本卷规定 |
|---|---|
| 覆盖 | 以北宋建国（960）至元亡（1368）为主；为交代政权形成，辽上溯契丹建国及五代关系，西夏上溯党项定难军，金上溯女真诸部，蒙古上溯成吉思汗统一诸部（1206）。南宋亡（1279）后仍书写元朝南方、东北、西北和海域治理。辽亡后的西辽（1124—1218）、金—蒙古转换、西夏亡（1227）及元初征服诸区域均不作为附注处理。 |
| 并立原则 | 宋、辽、西夏、金、元及其附属、竞争政权是同时行动者，不以宋朝年号或后世“大一统”叙事作唯一时间轴。每条账本事件保留该政权原始年号、干支、月日和换算依据；跨政权事件列出双方或多方纪年。 |
| 账本用法 | 每项先用可回查的编年、纪传、实录残卷或同代文书确立日期/相对序列，再按问题加入制度、财政、地方、宗教、碑刻、考古或外部材料。`core_sources` 写明书名、卷/篇；文书写编号、语言和出土地；碑刻写碑名、地点和释文版本；发掘材料写报告、遗址单位或图版。`source_caveat` 必须说明成书距离、文体、立场和主要冲突。 |
| 版本与检索 | 正史优先采用有校勘记的中华书局点校本，必要时核对《四库》、百衲本或相关校勘成果；《宋会要辑稿》须写明门类、卷次与辑本性质。契约、文书、碑刻、钱币和考古均引正式整理本或发掘报告及释读版本。数据库、影印页面和地理信息库只作检索入口，不替代可核版本。 |
| 不能直接推出 | 官修国史的正闰、夷夏、僭伪和亡国标签不能单独决定合法性、族群或动机；诏令、会要、律令和条格只能证明规范或议论，不能单独证明全国执行；户口、田亩、岁入、兵额、战果和朝贡数字须保留统计口径与政治场景；一座墓、一批文书或一处窖藏不得外推为全国社会。 |

## 一、编年基线与跨政权校核

| 《建炎以来系年要录》 | 1127—1162；高宗朝至绍兴末 | 李心传南宋编年，依日历、实录、时政记和文集，详于建炎南渡、宋金战争、和议及南宋中枢；现存传本、删节与编者立场影响尤须注意 | 南宋初年逐日/逐月序列，核查和战、行在、财政动员与官员活动；1162 年后不可外推，宋金双方战报要并列 |
| 《宋会要辑稿》 | 宋代全期，按门类保存政令与机构材料 | 徐松据《永乐大典》辑出，保留大量原会要佚文；条目常残缺、错简、日期不全且经辑佚重组，反映行政记录和后设分类而非连续事实 | 以食货、职官、兵、刑法、蕃夷、礼等门类追踪制度提出、反复申令和术语；必须区别颁行、复申、局部实施和后来的汇编题名 |
| 《皇宋十朝纲要》《隆平集》《国朝会要》残存与《宋大诏令集》 | 北宋为主 | 编年、政书和诏令汇集，部分更接近宋代档案传统，但传本、题署、编纂层次不一 | 补《长编》残缺，追查诏令与官制条目；逐条核对成书年代和是否为后出重编 |
| 《三朝北盟会编》《建炎以来朝野杂记》《四朝闻见录》等宋人/宋末史料 | 1115—13 世纪，宋金冲突和南宋政治为重 | 史料汇编、笔记与见闻保存细节和异说，也有追责、门户与传闻成分 | 为靖康、和战、使节、俘虏、军民经验建立候选证据链；不得以笔记单独确证日期、动机或军数 |
| 《资治通鉴》、胡三省《音注》及《续资治通鉴》 | 前者至 959；后者覆盖宋元但为清代毕沅续编 | 前者是 960 年以前转折的连续前史；后者便于检索而非同代证词。两者都以中原政权、治乱和鉴戒组织史事 | 五代—宋转折的背景和跨朝检索；宋元事件回查其更近的正史、编年史或文书，不以续编定案 |
| 年号、干支、历谱、天象、碑刻纪日与日历研究 | 全期，特别是改元、并立政权和蒙元诸历并用处 | 可发现误月、误置、同名年号与历法换算问题；天象本身也服务灾异、合法性和纪功语言 | 统一日期字段并记录换算依据；有异文、闰月或换算争议时写“约”，不得由天象校验直接推出政治因果 |

## 三、正史、编年与制度材料的核心矩阵

| 材料 | 最强问题域 | 使用边界与交叉办法 |
|---|---|---|
| *Songshi*（《宋史》） | 帝纪提供宋朝政治和大事框架；志、表、列传涉及食货、兵制、职官、地理、河渠、礼乐、学校、选举、艺文与宋辽夏金关系 | 元末修史的速度、宋亡后的档案选择和正统史观使细节须核。尤其财政数字、战果、党争人物评语及“夷狄”分类，分别以《长编》《要录》《会要》、诏令、文书和对方材料校验；不可把《宋史》叙述当作辽夏金的自我记录。 |
| *Liaoshi*（《辽史》） | 辽帝纪、百官志、营卫志、兵卫志、食货志、地理志、国语解和列传；契丹与宋、高丽、女真关系 | 元修文本受保存材料和后设王朝框架限制；营卫、部族、北南面官名的沿革、覆盖范围及实际运行不能只据志书。将契丹文字题记、辽墓志、宋使行录、高丽材料和考古分期并列，保留地名、官号、族名的释读不确定性。 |
| *Jinshi*（《金史》） | 金本纪、食货/兵/百官/地理等志、交聘表、列传，及灭辽北宋、宋金战争、迁都和末期抗蒙 | 元修金史并非金朝档案的完整替代，宋金敌对叙述和官修合法性语言相互放大。用宋方编年、金代碑志、文集、诏令、考古和蒙古/高丽材料辨析日期、战役规模、税制和地方控制。 |
| *Yuanshi*（《元史》） | 本纪、百官志、食货志、河渠志、兵志、释老志、地理志及列传；元朝统治、征服、行省和对外往来 | 明初仓促纂修虽保存实录条目，亦有纪年、篇章和姓名异同；“元朝”不覆盖所有蒙古汗国。以《元典章》《通制条格》、实录残文、碑刻、文书及蒙古、波斯、高丽材料交叉；对行省、色目、纸钞和灾害数字记录来源、口径与地域。 |
| *Xu Zizhi Tongjian Changbian*（《续资治通鉴长编》） | 北宋政治、诏令、任免、财政与边事的高密度编年，且保存宋实录等佚书线索 | 它是南宋史家重组的宋廷材料库，不是逐日原始档案；章节残缺、引用层次、李焘的评断及中枢视角均须说明。与《宋会要辑稿》及地方/辽夏材料互证，不能用作辽或西夏社会的唯一来源。 |
| *Jianyan yilai xinian yaolu*（《建炎以来系年要录》） | 1127—1162 的南宋初创、宋金战争、和议、行在和中枢人事 | 对绍兴朝尤重要但止于 1162，今本传承和编者的政治判断须查明；宋方诏奏、战报和使节材料要同金方记录及碑刻、地方材料分列，不以“和”“战”标签压缩不同地区经验。 |
| *Song Huiyao Jigao*（《宋会要辑稿》） | 职官、食货、兵、刑法、礼、学校、蕃夷、方域等门类的制度条目、诏令和反复施行记录 | 是徐松自《永乐大典》辑出的残存会要而非原会要全貌；错简、缺文、编年断裂、后设门类和抄录误差都可能存在。它适合追索制度文本和“再申令”，不自动证明实施、财政总额或全境一致。 |
| 《宋会要》外的会要、政书与法令 | 《庆元条法事类》、宋代敕令残本，《元典章》《通制条格》及《大元圣政国朝典章》类材料 | 法典、条格和行政汇编最能说明规范语言、适用对象和机构术语；须先辨版本、颁行年代、是否为重修或摘录，再以判牍、地方文书、碑刻和账册式材料检查实际执行。 |
| 元朝实录残文、《元朝秘史》与波斯文史籍 | 蒙古征服、皇族网络、草原政治、对西北和欧亚世界的观察 | 汉译/抄本、成书层次、口述传统、赞颂目标与译名对应均须单列。《元朝秘史》不可直接覆盖元代后期制度；拉施特丁等波斯文叙事亦有宫廷来源和时间距离，须标明语言、版本和观察位置。 |

## 四、按问题调取的材料

| 问题层 | 优先材料 | 可回答的问题 | 偏向、限制与账本接入 |

## 六、进入事件账本的操作规则

| 事件类型 | 最低来源组合 | `source_caveat` 应写明 |
|---|---|---|
| 政权建立、继承、改元、都城迁移与条约 | 一条主体编年/本纪，加对方或独立编年、碑刻纪日或文书之一 | 双方纪年与称谓；正统/僭伪标签；日期换算、条约文本存佚和结果是否实际落实 |
| 制度、法律、官制与行省/路州县变动 | 正史志或会要/条格的具体条目，加诏令、任官记录、判牍或地方材料之一 | 条文、颁布、复申、实施和废弛的区别；适用地域、人群和后出汇编层次 |
| 财政、赋役、货币、市场与贸易 | 食货志/会要/政书或档案性材料，加地方文书、考古或另一政权材料 | 数值口径、币制、实物单位、统计机关、地域、年份和政治场景；样本是否可比 |
| 战争、征服、外交与边疆 | 各方至少一条叙事或编年材料；可加碑刻、路线/城址或第三方材料 | 敌我称谓、军数和战果的报告性质；地点识别、信息距离、双方日期与“控制”含义 |
| 宗教、地方社会与文化事件 | 同代题记、文书、碑志或可靠文本，加制度/传记/考古的另一类材料 | 成书、抄写、雕印、立碑和事件日期的区别；赞颂、谱系、神异或精英自述的文体目的 |
| 只有单一来源的候选事件 | 允许入账以保持时间线，但标为低置信度并说明仅存来源 | 不将推测补写为事实；保留原文措辞和待查方向，不衍生未经证实的因果链 |

## 七、覆盖空白与必须显式保留的争议

| 主题 | 不可预设的结论 | 最低处理标准 |
|---|---|---|
| 宋、辽、夏、金、元的合法性 | “正统”“僭伪”“夷夏”即为中性政治描述，或任一王朝年号可自然覆盖其对手 | 使用当时原称和后出史书标签时注明出处；分开法律名义、贡赐关系、军事实控、行政征收和地方认同。 |
| 宋代财政与“积贫积弱” | 食货志与会要数字足以证明单线繁荣或衰弱，制度名称即等于实际能力 | 对每个数字记录口径与地区；将中央预算、地方征收、军费、货币、市场和家庭负担拆开，特别防止江南材料外推。 |
| 辽夏金的国家与族群 | 契丹、党项、女真、汉人、色目等是稳定且内部同质的现代民族类别；采用汉官制即为“汉化” | 保留史料原词，区分政治归属、法律身份、语言文字、婚姻、职业、地域和自称；不以姓名、服饰、官号或一条碑文单独定性。 |
| 西夏史料失衡 | 宋、辽、金、元史中关于西夏的战报和朝贡记录即可替代西夏主体材料 | 优先加入西夏文文书、印本、碑刻和考古，并标注释读阶段；确无本方材料时明确“外部观察”，不补造财政、人口或动机。 |
| 宋金战争与靖康/南渡 | 宋方受难或金方胜利叙事可单独说明事件；俘虏、伤亡和人口数字可直接相加 | 对照宋、金及第三方记录，标注叙事位置和战时宣传；将朝廷、军队、地方和流民的不同经验分开。 |
| 蒙古征服与元朝形成 | 蒙古征服是单一、同步过程，或《元史》即可说明整个蒙古帝国 | 区分大汗政权、各汗国和被征服地区；同时使用汉文、蒙古文、波斯文、高丽等材料，注明翻译、地名和时间体系的差异。 |
| 元代身份、税役与纸钞 | “四等人制”或单一户籍分类足以解释法律与社会实际；发行额可等同流通或物价 | 区分条格、官文书和地区实践；数字附币种、兑换、地域和时段。说明行省、军户、匠户、站户、寺院与城市居民的材料不能互换。 |
| 人口、土地、灾害与环境 | 户口、垦田、河患、旱涝和粮价可直接重建全域人口或因果关系 | 保留记录机构、观察地点和灾异修辞；以多地文书、考古和气候/环境研究作限定，不把相关性写成单因解释。 |
| 海贸与跨区域交流 | 朝贡册封、外销器物或一艘沉船即可证明国家控制、固定航线或文化单向传播 | 记录货物年代、生产地、发现地、运输与消费之间的推断环节；将民间、官方、宗教和侨民网络分开。 |
| 地图与疆界 | 正史地理志、战争叙事或现代地图可给出精确稳定国界 | 地图只标示约略控制、路线或争议区，并依具体年份注明证据；对羁縻、互市、驻军、行政设置和实际征收分别表达。 |

## 八、交接边界

本文件只建立来源池、问题矩阵和账本取证规则，不创建事件账本，不写正文、地图、PDF 或任务队列。后续 Chronology Agent 应先以本矩阵所列主体材料建立可审查的事件池；每一项事件仍需逐条核验，不能将此处“可用材料”视为已确认的史实。
