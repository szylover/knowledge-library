# 《中国大事编年》书系规范

## 定位

这是一个多卷、可并行生产的通俗中国史书系。它以可靠编年为骨架，但同时写政治、制度、经济、社会、军事、边疆、文化、宗教与思想。它要有史书的叙事力量，不是皇帝年表、资料卡合集或虚构历史小说。

## 分卷

| 卷 | 范围 | 核心问题 |
|---|---|---|
| 第一卷 | 西周至秦 | 封建网络、诸侯国家、国家化和统一帝国 |
| 第二卷甲 | 汉帝国（前206—220） | 帝国调适、财政边疆与两汉危机 |
| 第二卷乙 | 三国（220—280） | 区域国家、战争补给与再统一 |
| 第二卷丙 | 两晋南北朝（280—589） | 门阀、迁徙、部族政权与再统一 |
| 第三卷 | 隋唐五代 | 再统一、世界帝国、藩镇与财政转型 |
| 第四卷 | 宋辽夏金元 | 多政权、商业财政、边疆和征服帝国 |
| 第五卷 | 明 | 皇权、白银、赋役、边疆与晚期危机 |
| 第六卷 | 清至1912 | 征服王朝、近代冲击与帝国终结 |

后续可增加土地税收、经济社会、文化思想、地图和史料的专题卷；专题卷不能替代各历史卷的叙事。

## 内容契约

每卷同时写六层：编年政治、制度/地方权力、经济财政、社会人口、军事边疆、文化思想。每卷还必须包含国家 dossier、跨政权专题、经济社会专题、文化思想专题、制度得失、地图与索引。

事件先进入账本，再进入正文。账本字段为：

```text
internal_id, date/reign_year, polity, event, confidence,
core_sources, source_caveat, modern_research_direction,
causes, consequences, target_file, tier
```

事件分 A/B/C 级：A 级多页叙事，B 级完整条目，C 级保证时间和因果连续。卷一目标账本为 1,300–1,900 条。

## 史料契约

不建立死板的来源等级。按事件性质选择并交叉：

- 《史记》是周秦卷核心叙事；二十四史是后续卷的核心主体。
- 《资治通鉴》与续编提供编年脊柱，不是唯一事件库。
- 实录、志书、列传、国别材料、地方文书、简牍、金文、墓葬、诗文、宗教材料按问题使用。
- 吕思勉用于社会经济文化长线；钱穆《国史大纲》《历代政治得失》用于时代大势与制度得失的提问，不替代原始材料。
- 史料不一致时写出分歧。现代研究只概括和标注，不大段复制。

## 文学契约

- 从一件可见之物、一次出奔、诏命、会盟、调兵、城门、道路、河流或战场进入。
- 人物是受资源、信息、联盟、家族和制度约束的行动者；写选择与代价，不贴道德标签。
- 借鉴《左传》《史记》《资治通鉴》和正史的动作推进与前后相承，但所有 prose 原创。
- 不虚构对话、内心、服饰和未被材料支持的场景。
- 古文窗口仅在真正照亮事件时使用：短原文 + 出处 + 语境 + 文体/立场；不逐句翻译。

## 地图与视觉契约

- 暖纸色、朱褐、铜色；不用科技蓝。
- 地图是第二叙事线：每时代总览、国家位置、战争/迁徙/道路/边疆局部图。
- 所有图表独立位于 `figures/<period>/`，正文只 `\input`。
- 每张图标“示意/疆界约略”，不伪造古代精确国界；不搬运无明确许可的网络地图。

## 多 Agent 工作流

| 角色 | 输出 |
|---|---|
| Chronology | `data/event-ledger/` |
| Source | `data/source-corpus/` 和来源注记 |
| Narrative | 一个锁定的国家/专题 `.tex` 范围 |
| Cartographer | 独立 `figures/` 文件 |
| Indexer | 可跳转事件/国家/主题/来源/地图索引 |
| Integrator | wrapper、主文件、编译、PDF、进度、合并 |

所有任务在 `WORK_QUEUE.yaml` 中声明。Agent 必须使用独立 worktree 和 `chronicle/<task-id>` 分支；一个文件只能有一个 writer。Integrator 是唯一可以合并至 `main` 和发布根目录 PDF 的角色。

详见系列 `AGENTS.md` 与各卷 `AGENTS.md`。

## 发布契约

每次 Integrator 合并并更新 `pdf/china-chronicle/` 中任一卷 PDF，必须将整个目录同步到 Azure Blob Storage 账户 `szydownloads` 的容器 `downloads`、路径 `books/china-chronicle/`。下载页使用直接 Blob 链接 `https://szydownloads.blob.core.windows.net/downloads/books/china-chronicle/volXX-<slug>.pdf`；发布后须确认对应文件可访问。书系不再重复同步到 `$web` 静态网站容器。
