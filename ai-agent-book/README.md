# 《从零到一：AI Agent 工程师转行与面试完全指南（2026）》

> 面向传统软件工程师的 AI Agent 转行路线图

---

## 关于本书

本书是为**想从传统软件工程（前端/后端/全栈/测试/运维）转行到 AI Agent 方向**的开发者编写的系统性指南。

2026 年，AI Agent 已从实验室走向生产环境。Gartner 预测到 2028 年 33% 的企业软件将集成 Agentic AI。然而，大多数转行指南要么过于理论，要么停留在调 API 的表面。本书试图在**深度理解**和**实战可用**之间找到平衡：

- 不仅告诉你"什么是 RAG"，还教你如何设计一个生产级 RAG 系统
- 不仅列出面试题，还给出评分标准和深度解答
- 不仅介绍框架，还横向对比帮你做技术选型

**目标读者**：有 2+ 年软件开发经验，熟悉至少一门编程语言（Python 优先），想系统性转入 AI Agent 领域的工程师。

## 适合读者

- 想从传统软件开发转行到 AI 方向的工程师
- 准备 AI Agent 相关岗位面试的求职者
- 对 AI Agent 技术栈感兴趣的独立开发者
- 希望系统了解 AI Agent 生态的技术管理者

## 技术前提

- 熟悉至少一门编程语言（Python 最佳）
- 了解基本的 Web 开发概念（HTTP、REST API）
- 有基础的数学素养（线性代数、概率论了解即可，本书不要求深入）
- 不要求有 ML/DL 经验（本书从零讲起）
