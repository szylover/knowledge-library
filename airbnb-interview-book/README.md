# Airbnb 面试通关手册

一本面向 Airbnb（及同类一线公司）SWE 面试的中文备战手册，**重点不在于堆答案，而在于把“怎么想”讲清楚**——每道题都带可迁移的设计思想。

## 内容结构

- **第一部分 · 面试全景**：流程、各轮权重、Airbnb 文化价值观、时间线。
- **第二部分 · 算法编码**：Airbnb 算法心法（可扩展代码 / 迭代追问 / 面向对象建模）+ 高频题逐题精解（题意、思路、设计思想、C++ 实现、复杂度、面试官追问）。
- **第三部分 · 系统设计**：通用方法论 + 订房 / 搜索 / 消息 / 支付逐题精讲 + 可复用的分布式设计思想（幂等、Saga、缓存、分片、限流、一致性）。
- **第四部分 · 行为面与冲刺**：Airbnb core values 对应的 STAR 故事框架 + 两周冲刺计划 + 免费资源清单。

## 排版特色

正文用四种彩框承载“设计思想”这条主线：

| 框 | 含义 |
|----|------|
| 💡 核心思想 | 这道题/这个系统真正考的设计心法 |
| ⚠ 常见陷阱 | 最容易写错/漏掉的边界 |
| ↪ 面试官追问 | 现场常见的 follow-up 与应答 |
| ✓ 要点 | 复杂度与速记结论 |

## 构建

```bash
# WSL / Linux（推荐）
make              # 两遍 xelatex
# 或
make tectonic     # 用 tectonic，无需本地 texlive
```

等宽字体使用 **DejaVu Sans Mono**（渲染代码与框图），需预先安装到系统字体。

产物：`main.pdf`。
