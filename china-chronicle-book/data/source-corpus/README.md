# 史料池

每一卷先建立史料池，再从史料池抽取事件进入 `data/event-ledger/`。本目录记录来源覆盖范围、材料性质、可用于确认的事实类型、争议与账本进度。

## 流程

```text
基础编年文本 / 正史 / 实录 / 出土材料 / 现代研究
                         ↓
                来源与年份交叉核对
                         ↓
                event-ledger（事件账本）
                         ↓
          国家 dossier / 跨国合卷 / 经济文化专题
                         ↓
                       PDF
```

一条事件不因出现在名著中就自动进入正文。它需要有明确的日期或相对顺序、可说明的行动者与后果，并标明来源的文体和可信度。
