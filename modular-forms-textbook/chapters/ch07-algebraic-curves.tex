% ============================================================================
\chapter{模曲线作为代数曲线}
\label{ch:algebraic-curves}
% ============================================================================
% 对应 Diamond \& Shurman Chapter 7

前一章我们已经把 $X_0(N)$、Hecke 算子与 Jacobian 压缩进同一个复几何框架里；
本章要做的，是再向算术方向迈出决定性的一步：
把模曲线从紧黎曼面升级为定义在 $\Q$ 上的代数曲线，
并进一步说明它在整数环与模 $p$ 上如何延拓。

本章结构如下：
\begin{itemize}
  \item \textbf{§\ref{sec:ch07-motivation}} 解释为什么仅有复解析结构还不够，必须寻找 $\Q$-模型。
  \item \textbf{§\ref{sec:ch07-algebraic-varieties}} 补足后文所需的最小代数曲线语言。
  \item \textbf{§\ref{sec:ch07-modular-curves-over-q}} 用 $j$-函数与模解释构造 $X_0(N)$ 的 $\Q$-模型。
  \item \textbf{§\ref{sec:ch07-hecke-correspondences}} 把 Hecke 算子改写成代数对应。
  \item \textbf{§\ref{sec:ch07-modular-interpretation}} 说明整模型、好约化与半稳定约化的意义。
  \item \textbf{§\ref{sec:ch07-exercises}} 用显式计算巩固模多项式、Hecke 对应与约化理论。
\end{itemize}
