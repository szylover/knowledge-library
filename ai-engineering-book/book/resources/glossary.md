# 术语表 Glossary

> 随全书累加。术语首次出现的章节会给出更完整解释，此处仅作速查。

| 术语 | 说明 |
|------|------|
| Token | LLM 处理文本的最小单位，计费与上下文长度均以 token 计。 |
| Context Window | 模型单次可处理的最大 token 数（输入+输出）。 |
| RAG | Retrieval-Augmented Generation，检索增强生成。 |
| Embedding | 把文本映射为稠密向量，用于语义检索。 |
| MCP | Model Context Protocol，标准化的工具/数据源接入协议。 |
| Function / Tool Calling | LLM 输出结构化调用请求，由宿主执行外部工具。 |
| Guardrails | 对输入/输出施加的安全与合规约束。 |
| Re-ranking | 对初检结果用更强模型重新排序以提升相关性。 |
