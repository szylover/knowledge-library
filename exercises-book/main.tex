\chapter*{前言}
\addcontentsline{toc}{chapter}{前言}

本习题集是《模形式与费马大定理》主教材的配套练习册。
主教材每章侧重定理的完整证明与直觉建立；
本册则把重心放在**动手计算**和**证明演练**上。

\medskip
\textbf{如何使用本书。}
每章习题分为三个梯度：
\begin{itemize}
  \item \easy\ \textbf{基础题}：直接应用定义和基本公式，做完后应能独立重述相应定义
  \item \medium\ \textbf{进阶题}：需要组合本章多个结果，或与前一章的内容交叉
  \item \hard\ \textbf{挑战题}：需要创造性想法，部分题目接近研究水平
\end{itemize}

建议先独立做题，写出完整论证，再翻看解答。
每章末尾给出所有题目的完整解答。

\medskip
\textbf{符号约定。}
与主教材一致：$\HH$ 表示上半平面，$\SL_2(\Z)$ 是模群，
$\Mk(\Gamma)$/$\Sk(\Gamma)$ 分别是权 $k$ 模形式/尖点形式空间，
$\chi_\ell$ 是 $\ell$-进分圆特征，$\Frob_p$ 是算术 Frobenius。
